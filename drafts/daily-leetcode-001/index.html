917 Reverse Only Letters

Link: https://leetcode.com/problems/reverse-only-letters/

这道题不难，只需要翻转字母的字符，那么其实只需要使用一头一尾两个指针依次往里靠寻找为字母的字符，然后遇到后便翻转即可。

class Solution {
public:
    string reverseOnlyLetters(string S) {
        int i = 0, j = S.length() - 1;
        string ret(S);
        
        while (i < j) {
            while (!isalpha(ret[i]) && i < j)
                i++;
            while (!isalpha(ret[j]) && i < j)
                j--;
            if (i < j) {
                swap(ret[i], ret[j]);
                i++;
                j--;
            }
        }
        
        return ret;
    }
};
